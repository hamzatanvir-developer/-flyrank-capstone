# FlyRank Capstone Project

This repository contains my capstone project for the FlyRank internship track. The project will demonstrate practical software development, AI-assisted development, version control, and deployment practices.

## Project Status

The project is currently in the setup phase.

## Planned Technology Stack

* Node.js
* JavaScript or TypeScript
* Git and GitHub
* Google Antigravity
* AI-assisted development tools

## Project Goals

* Build a complete and maintainable capstone application
* Follow clean coding and documentation practices
* Use AI responsibly throughout the development process
* Maintain a clear Git history using Conventional Commits
* Document the setup, development, and deployment process

## Getting Started

Detailed installation and development instructions will be added as the project develops.

## Development Workflow

This project follows the Conventional Commits specification.

Examples:

```text
feat: add a new feature
fix: resolve an application issue
docs: update documentation
chore: perform repository maintenance
```
